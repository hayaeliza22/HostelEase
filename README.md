Hostel Ease – Smart Hostel Management System

Project Description
-Hostel Ease is a web-based hostel management system that streamlines communication between students and wardens. Students can submit complaints, apply for leave, send anonymous suggestions, and raise SOS alerts. Wardens can monitor requests and update statuses in real time for faster issue resolution.

Tech Stack
-Python (Flask)
-HTML5
-CSS3
-JavaScript
-SQLite
-Gunicorn (Deployment)

Features
-Complaint submission and tracking
-Leave request approval system
-Anonymous suggestion portal
-SOS emergency alerts
-Real-time status updates
-Warden dashboard analytics
